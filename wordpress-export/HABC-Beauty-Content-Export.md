## WORDPRESS TECHNICAL REQUIREMENTS

### Theme Features Required
1. Custom Post Type: "Services"
2. ACF (Advanced Custom Fields) integration
3. Image upload capabilities throughout
4. Mobile-responsive design
5. Video background support
6. Shopify integration
7. Review aggregation (Google/Facebook)
8. Booking widget integration (Vagaro/StyleSeat)

### Editable Elements via WordPress Admin
- All page titles and subtitles
- Hero video/image backgrounds
- All service details (name, description, price, image)
- Featured services on homepage
- Team member photos and bios
- Contact information
- Social media links
- Footer content

### Color Scheme
**Primary:** Black (#000000)  
**Accent:** Peach Cream (#FFE5B4)  
**No other colors should be used**

### Typography Recommendations
- Elegant serif font for headings (e.g., Playfair Display)
- Clean sans-serif for body text (e.g., Inter, Montserrat)
- Italic styling for brand name and section titles

1. Service images (each service needs 1 image minimum)
2. Team member photos (2 photos of Caroline)
3. Gallery images
4. Hero background video/images
5. About page images
